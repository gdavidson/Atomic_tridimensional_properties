Pour lancer le logiciel: executez "Run3DProperties.jar" dans le répertoire /dist.

Prérequis: JRE 7.

Cliquez sur "calculer les propriétés" puis sélectionnez le fichier pdb adéquat (un reminder se situe sur la fenetre du FileChooser).
Les fichiers de sortie sont écrits dans le répertoire courant (probablement /dist).
